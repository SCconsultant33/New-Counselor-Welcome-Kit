# Oakland Schools New Counselor Survival Guide

A static, GitHub Pages-friendly prototype that reworks the current New School Counselor Welcome Kit into a responsive field-guide style web app.

## Local Preview

From this folder, serve the files with any static server. For example:

```powershell
python -m http.server 5173
```

Then open `http://localhost:5173/`.

## Prototype Notes

- Built as dependency-free HTML, CSS, and JavaScript for the first clickable mockup.
- Content is based on the 2025-2026 PDF welcome kit.
- Visual styling uses the CIA/Oakland Schools style guide as the foundation.
- The structure is ready for GitHub Pages because `index.html` lives at the repository root.
